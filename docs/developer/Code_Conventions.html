Cayambe - Code Conventions
In addition to the normal Java Code Conventions Cayambe has added a few notes here that will help us write and read the Cayambe code the same.

Class Code Convention - This is the order that method and vars should be in within your classes

public static final vars
public vars
protected vars
private vars

(Business Methods)
static methods
public methods
protected methods
private methods

(Container Methods)
static methods
public methods
protected methods
private methods


Class Naming - This is where you should put your classes

org.cayambe.client - client access classes
org.cayambe.ejb - EJB classes
org.cayambe.util - Utilities
org.cayambe.dao - Data Access Objects
org.cayambe.core - The center of it all
org.cayambe.web - Web related classes
org.cayambe.web.action - Struts classes
